## 🔍 Descrição do Projeto


A ideia do projeto é unificar o conhecimento que ambos tem sobre visão computacional, para utilizar um método de classificar imagens com base em features gerados por descritos como BoVWs. Então esse projeto foi divido em duas sessões, a extração de descritores das imagens realizado por @Joaquim e a classificação
utilizando o classificador Support Vector Machine (SVM), trabalho realizado por @JhemersonLincon.
