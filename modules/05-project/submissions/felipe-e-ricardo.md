## 🔍 Descrição do Projeto

O projeto é uma integração do PySUS para coleta de dados e PyQt5 para visualização de gráficos com esses dados, 
contando com o Poetry para gerenciamento de dependências, orientação a objetos e uam arquitetura robusta com inversão de dependência.
Com o intuito de melhorar o aprendizado em Git, todo o desenvolvimento foi feito utilizando as boas práticas aprendidas no Git Roadmap.

## 📚 O que Aprendemos

- Felipe: Antes do Roadmap eu até sabia um pouco de Git, mas nada sobre branches, agora, se criar, excluir, fazer merge, resolver conflitos e abrir pull requests, tudo isso com boas práticas.
- Ricardo: Eu aprendi a utilizar branch da forma como as organizações utilizam, entendi como os diversos métodos de colaboração que existem e como cada um se adapta melhor para cada caso.
