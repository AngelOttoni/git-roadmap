## 🔍 Descrição do Projeto

Este projeto demonstra, de forma simples e didática, como integrar aplicações em Python a Modelos de Linguagem de Grande Porte (LLMs) por meio de APIs de IA Generativa. Ele cobre a obtenção e uso de chaves de API (OpenAI e Google AI Studio), o envio de prompts e a recepção de respostas geradas (texto/código), evidenciando como essas interfaces facilitam a incorporação de IA em diferentes soluções. Sarah concentrou-se na integração com a OpenAI API e na estrutura do script de execução (CLI), enquanto Tamires organizou o repositório, documentou o passo a passo de execução e validou o fluxo de autenticação com as chaves (incluindo a preparação para uso do Gemini).
