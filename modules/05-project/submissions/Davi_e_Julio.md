## 🔍 Descrição do Projeto

O Invincible-TicTacToe é um jogo da velha (Tic Tac Toe) com interface gráfica desenvolvida em Python utilizando PyQt5. O projeto permite que o usuário jogue contra uma inteligência artificial baseada no algoritmo Minimax, garantindo partidas desafiadoras.  
A estrutura do projeto foi dividida entre lógica do jogo, interface gráfica e implementação da IA.  
- **Davi Santos:** Implementou a lógica do jogo, desenvolvimento e integração com a IA.
- **Nome 2:** Desenvolveu a interface gráfica com funcionalidades para maior fluidez do jogador.

O projeto se relaciona com conteúdos do roadmap como algoritmos de busca, controle de versão e desenvolvimento de interfaces gráficas.
