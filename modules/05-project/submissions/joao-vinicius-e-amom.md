### 🔍 Descrição do Projeto

O projeto consiste em uma aplicação de gerenciamento de tarefas com arquitetura cliente-servidor. Desenvolvemos um **servidor HTTP nativo em Python** (sem frameworks externos) que gerencia o CRUD de tarefas via JSON. O sistema conta com persistência de dados local e tratamento de caminhos (OS path) para garantir compatibilidade entre Windows e Linux. Seguindo o Git Roadmap, o desenvolvimento foi organizado em branches, com foco em commits semânticos e integração entre as camadas de front e backend.
